## 1.0.2
### Changed
- Renamed main class, split the code and tested for v1.22.3

## 1.0.1
### Changed
- Cleaned up source code for the public GitHub repository.
- Renamed several internal variables for better readability.

## 1.0.0
### Added
- Initial release of the mod on Vintage Story ModDB (built with .NET 10 against game version 1.22.2).
- Protection from falling blocks using armor, featuring two different modes and a configuration file:
  - **Lottery Mode**: Mirrors native vanilla combat mechanics.
  - **Layered Mode**: Applies protection sequentially from top to bottom.
- Optional logging setting for damage calculation debugging.
